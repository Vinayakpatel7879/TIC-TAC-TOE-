

# Tic-Tac-Toe Game

A classic, interactive, and responsive **Tic-Tac-Toe game** built from scratch using clean frontend web technologies. This project features an intuitive UI/UX, robust game logic, and adaptive styling for a seamless player experience on both desktop and mobile devices.

---

## 🚀 Features

* **Two-Player Gameplay:** Seamlessly alternates turns between Player X and Player O.


* **Real-Time Win Detection:** Instantly evaluates matches against standard winning combinations to crown the winner.


* **Draw Validation:** Smart move-tracking automatically detects and declares a tie if all 9 cells are filled without a winner.


* **Interactive UI Animations:** Smooth transitions that toggle modal screens for a clean game loop.


* **Responsive Grid Layout:** Styled entirely using CSS flexbox, grid dynamics, and viewport units (`vmin`), making the game board scale flawlessly across different screen resolutions.


* **Instant Reset Capabilities:** Offers quick action items with "Reset Game" and "New Game" controls to clear the board and start fresh at any time.



---

## 🛠️ Built With

* **HTML5:** Structures the layout, grid components, and modal interfaces `index.html`.


* **CSS3:** Delivers custom color palettes, shadow elements, and a mobile-friendly responsive canvas `style.css`.


* **JavaScript (ES6):** Handles event manipulation, DOM updates, and core mathematical win condition logic `app.js`.



---

## 🕹️ How to Play

1. Open the game in your browser via `index.html`.


2. Player O starts the game by clicking any grid cell, followed by Player X.


3. The first player to align 3 matching symbols horizontally, vertically, or diagonally wins.


4. Click **New Game** or **Reset Game** to instantly challenge your opponent to a rematch!
